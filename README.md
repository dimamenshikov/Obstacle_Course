2. Жанр: Third-Person
3. Платформа: ПК
4. Управление: WASD и мышь
5. Цель уровня: Найти и вынести древний кристалл солнца с острова
6. В игре присутствует возможность настроить качества графики

Элементы геймплея

1. Перемещение персонажа:
    1. Игрок управляет персонажем с помощью клавиш WASD для передвижения
    1. Игрок управляет камерой с помощью мыши для обзора окружения
2. Система здоровья: У персонажа есть здоровье, если оно кончится персонаж возродится в последней контрольной точке
3. Счётчик времени и очков: После победы лучший результат по времени и очкам записывается в рекорд
4. Интерактивное взаимодействие с окружением: В игре есть 4 локации с ловушками и интерактивными предметами - телепорты, трамплины, платформы, двери, которые активируются, если подойти к ним. В игре есть рычаги, которые активируются на E. В игре есть возможность на Q выключить свет 
<img width="792" height="506" alt="image" src="https://github.com/user-attachments/assets/e1afd380-6d42-4bf2-96b2-52cd9486d6a9" />

5. Логика бонусов: На уровне есть 3 вида бонусов:
    1. Жизни - увеличивает жизни
    2. Монеты - увеличивает очки
    3. Аптечка - увеличивает здоровье
<img width="792" height="506" alt="image" src="https://github.com/user-attachments/assets/71f64e98-c399-4902-9902-eb306e543041" />

6. Пушки: В каждой локации определённый вид пушки. 
Всего есть 4 вида пушек:
    1. Голубая - стреляет прямо-летящими ракетами.
    2. Синяя - стреляет ядрами, летящие с учетом гравитации.
    3. Красная - стреляет самонаводящимися ракетами.
    4. Зелёная - стреляет ядрами с учётом траектории движения игрока.
